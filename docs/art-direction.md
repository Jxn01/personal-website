# JXN-000 · ART DIRECTION REFERENCES

Reference imagery sent by Norbert (inline, in chat). This file records each
reference and — more importantly — **what we're extracting from it**. Per
Norbert's instruction: *"don't hyperfocus on them, just pick out the best
parts and work it into a coherent design"* and *"don't make the whole website

Standing rule: the JXN-000 spec's NO-list still governs. These references are
raw ore — we take **texture, palette tension, type energy, and humor
register**, never literal iconography (no boomboxes, graffiti fonts, artist
imagery on the site itself).

### B1-02 · "Boombap" graffiti hand-style tag (black on white)

  e.g. a hand-drawn signature/scribble in the liner-notes footer. One gesture,
  drawn once, never repeated. The human glitch in the system.

### B1-03 · Kendrick Lamar mixed-media poster
Mint/teal field, black stencil-cut display type top and bottom, portrait built
from collage: street-map texture filling the face, halftone dots, gold foil
scraps, scratch marks, denim texture.
**Extract:**
- **The portrait treatment.** Strongest single idea in the batch: a duotone
  portrait whose silhouette is filled/textured with *meaningful* material.
  For Norbert's photo: consider circuit-trace / city-grid / code texture
  resolving inside the silhouette (Budapest map texture is an option).
- Stencil-cut display energy for a few oversized words (hero scale) — sharp
  grotesque cut hard, not an actual stencil font.
- Controlled collage chaos: small scattered marks (×s, dots, dashes) as
  micro-texture — translates to our technical-label glyphs.
- Mint/teal × black × gold-foil: second strong vote for cyan-family accent
  with gold jewelry.

### B1-04 · B&W golden-era graffiti mural (illustration)
Dense monochrome wall: MC with mic, boombox, "BOOM BAP MAD FLOW / GOLDEN ERA /
BACK IN THE DAY" lettering everywhere.
**Extract:**
- One-color discipline at high density — a composition can be *loud* while
  using exactly one ink. Applies to our mono-label micro-copy layers.
- Background-wall energy: faint dense texture layers sitting far behind
  content (our dot grid / scanlines play this role).
- Nothing literal. No mics, no boomboxes, no slogan lettering.

### B1-05 · B&W photo: cat in bucket hat with chain, guy with boombox, NYC street
Documentary 90s-street photography style, heavy grain, deadpan.
**Extract:**
- **The humor register.** Completely straight-faced absurdity, zero winking.
  This is the site's voice in image form (the calculator that proves Euler's
  theorem lives in this register).
- B&W/duotone photographic grain treatment for photography.
- The chain = the gold accent philosophy: one glint of jewelry on an otherwise
  muted subject.

### Batch 1 — cross-cutting signals
1. **Palette:** repeated cyan/mint-family cold field × warm gold/mustard accents.
   Two of five references vote this way independently. (Final call still
   Norbert's, with batch 2 pending.)

5. **Humor:** deadpan, never explained.

## Palette (LOCKED)
| Token | Value | Role |
|---|---|---|
| `--bg-*` | #0a0a0c → #16161b | near-black base, never pure black |
| `--accent` | **#41EAD4** phosphor teal-cyan | THE cold accent. Machine-on. Terminal ink, links, live states |
| `--gold` | **#D8C08A** champagne | jewelry only: catalog numbers, link hover, one or two hairlines |
| `--cream` | **#EFE3C2** print cream | imagery & halftone layer — human warmth (the portrait, ASCII figure work) |
| `--error` | **#FF2E2E** signal red | system-state only: 404, errors. Never decorative |
| ink scale | #f4f4f6 → #55555d | text hierarchy |

Four registers: **cyan = machine on · gold = jewelry · cream = human/print ·
red = failure.** The spec's "ONE cold accent" rule holds — gold/cream/red are
utilities with strict jobs, not accents.

## Surfaces
1. **Sleeve** — paper-grain card, 1px border, mounted (never full-bleed),
   corner metadata stamp. Projects, portrait.
2. **Window** — OS chrome: title bar, controls. Terminal, /status, 404 dialog.
3. **Label** — NATO-stock `key: value` mono blocks. Metadata everywhere.
